
SIEM_ML_Project (Isolation Forest Demo)

How to run:
1. Create virtualenv and install packages: pandas, scikit-learn, joblib, streamlit, openpyxl
2. Run preprocessing: python src/preprocess.py
3. Train model: python src/train_iforest.py
4. Evaluate: python src/evaluate_model.py
5. (Optional) Visualize anomalies: python src/visualize.py
6. Run dashboard: streamlit run src/app.py
